# 04-01: Typography & Design Token Overhaul

## Objective

Replace the undefined `--font-terminal` / `--font-pixel` fallback chain with explicitly declared
font variables. Add Space Grotesk (readable sans) and JetBrains Mono (data/numbers) from Google
Fonts. Update the dark background tokens to the v3 indigo-black palette. Fix body font-family
and shrink the HUD player name from its current unreadable 32px VT323 size.

---

## Context

**Current state (verified from source):**
- `--font-terminal` and `--font-pixel` are used in 30+ places in styles.css but are **never
  declared** in `:root` — they fall through to the browser default (serif). The only font that
  actually loads is VT323 and Press Start 2P via Google Fonts.
- Body font-family: `'Segoe UI', 'Courier New', monospace` — reasonable fallback but not the
  retro-crisp aesthetic the game wants
- `.hud-player-name` is 32px VT323 — huge and hard to read at that resolution
- Background: `--bg-dark: #0d0d0d` (pure black), `--bg-panel: #1a1a1a` — flat gray stack; the
  v3 mockup uses deep indigo-black which is warmer and more distinctive

**Google Fonts import** (line 16 in index.html):
```html
<link href="https://fonts.googleapis.com/css2?family=VT323&family=Press+Start+2P&display=swap" rel="stylesheet">
```

## Tasks

### Task 1 — Update Google Fonts import in index.html

<read_first>
- `index.html` lines 13–17 (current font link tags)
</read_first>

<action>
Replace the existing Google Fonts `<link>` on line 16 of index.html with a combined import that
adds Space Grotesk (weights 400, 500, 600) and JetBrains Mono (weights 400, 500):

```html
<link href="https://fonts.googleapis.com/css2?family=VT323&family=Press+Start+2P&family=Space+Grotesk:wght@400;500;600&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet">
```

The existing `<link rel="preconnect">` tags on lines 14–15 stay as-is.
</action>

<acceptance_criteria>

</acceptance_criteria>

<read_first>
- `css/styles.css` lines 1–50 (`:root` token block)
</read_first>

<action>
