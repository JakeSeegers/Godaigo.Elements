# 04-02: HUD Shrine Progress Dots + AP Pip Bar

## Objective

Add two new visual elements to the HUD bar: (1) a row of 5 element-colored shrine progress
dots showing which elements the current player has completed, and (2) an AP pip bar showing
current action points as filled segments. Both are wired to the existing JS update paths so
they stay in sync with game state automatically.

<action>
**Step A — Add two helper functions** near the top of the HUD update section (after any
existing `updateHUD`-type function, around line 470):

```js
function updateShrineDots(playerIndex) {
    const scrollData = window.spellSystem?.playerScrolls?.[playerIndex];
    const activated = scrollData?.activated ?? new Set();
    ['earth', 'water', 'fire', 'wind', 'void'].forEach(el => {
        const dot = document.getElementById('shrine-dot-' + el);
        if (!dot) return;
        dot.classList.toggle('complete', activated.has(el));
    });
}

function updateApPips(apValue) {
    const max = 5;
    document.querySelectorAll('.ap-pip').forEach(pip => {
        const n = parseInt(pip.dataset.pip, 10);
        pip.classList.toggle('filled', n <= apValue);
    });
}
```
